# 高考志愿指导流程

## 1. 信息收集

必须确认：省份、年份、科类/选科、分数、位次。再确认偏好：985/211/双一流、公办/民办、城市/省份、专业方向、学费、是否接受中外合作/高收费/校区/冷门专业/调剂。

## 2. 家庭与职业约束

张雪峰式就业导向的第一步不是看学校，而是看家庭画像和职业出口：

- 家庭预算：是否能承受民办、中外合作、高收费、医学长周期、异地生活费。
- 是否支持读研：基础学科、医学、法学、生化环材等常有读研依赖。
- 是否急需本科就业：急需就业时优先就业出口清晰的工科、师范、财会、电力、计算机、电子信息等方向。
- 是否有行业资源：有资源可考虑家族行业或本地强关系行业；没有资源要更看重可迁移技能和校招机会。
- 是否接受异地就业：专业和城市产业要匹配，不能只看城市名气。
- 是否考公考编：关注法学、汉语言、财政、会计、师范、公安、马克思主义等岗位适配度。

## 3. 数据核验

按优先级核验：

1. 当年招生计划书、专业目录、院校专业组代码。
2. 省考试院一分一段表、批次线、投档线。
3. 高校招生章程和本科招生网。
4. 高校就业质量报告、教育部/官方学科信息。
5. 第三方平台，仅作辅助参考，必须标注非官方。

## 4. 位次优先

分数用于理解，位次用于匹配。每所院校至少比较近 2-3 年同省同科类/同选科/同批次的最低位次；若政策变化较大，降低历史数据权重。

## 5. 冲稳保分层

不要机械使用固定百分比。结合省份波动、招生计划变化、专业热度和用户是否接受调剂。

- 冲刺：能提升学校层级或城市，但历史位次通常优于考生位次，热门专业风险更高。
- 稳妥：历史位次与考生位次接近，有一定 buffer，专业选择不是全部热门。
- 保底：历史位次明显低于考生位次，招生计划稳定，专业和学费可接受。

## 6. 专业建议

区分“学校可进”和“专业可进”。推荐专业时说明：课程方向、适合人群、就业行业、城市产业、升学路径、证书/资格、专业录取风险、读研依赖、考公考编适配度。

## 7. 强取舍结论

最终不要只给表格，必须给清楚取舍：

- 学校优先版：冲学校层次，牺牲专业确定性。
- 专业优先版：保专业质量，可能降学校层级或城市。
- 就业优先版：普通家庭优先，突出本科就业、城市产业和校招机会。
- 最不建议方案：指出最大坑点，如高学费、冷门调剂、校区落差、专业就业路径不清。

## 8. 网页报告

网页报告结构：考生画像 → 家庭画像 → 筛选条件 → 张雪峰式就业导向结论 → 风险总览 → 冲稳保卡片/表格 → 专业红黄绿灯 → 专业详情 → 数据来源 → 免责声明。所有未核验字段显示“需人工核验”。
