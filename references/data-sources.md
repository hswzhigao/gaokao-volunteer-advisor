# 数据来源与引用规则

## 官方优先级

1. 省教育考试院/招生考试院：一分一段、批次线、投档线、招生计划。
2. 教育部阳光高考平台：招生章程、院校信息、专业目录入口。
3. 高校本科招生网：招生计划、历年录取、专业介绍。
4. 高校就业质量报告：就业去向、行业、薪资口径（如有）。
5. 教育部学科评估、双一流名单、软科等榜单：必须说明排名口径。

## 搜索关键词模板

- `{省份} {年份} 一分一段表 {科类}`
- `{省份} {年份} 本科批 投档线 {科类}`
- `{院校} {省份} {年份} 录取分数线 位次 {科类}`
- `{院校} 本科招生网 {年份} 招生计划 {省份}`
- `{院校} 招生章程 {年份} 体检 单科 外语 校区 学费`
- `{专业} 就业质量报告 薪酬 区间`（薪资只能辅助）

## 引用格式

每条关键数据后保留：

```text
来源名称，数据年份，适用省份/科类/批次，URL或出版物，访问日期 YYYY-MM-DD
```

## 数据状态

- 官方已核验：来自考试院、阳光高考、高校招生网/章程。
- 第三方参考：来自商业志愿平台、媒体、榜单；不得作为唯一依据。
- 需人工核验：找不到权威来源或存在冲突。

## 禁止

- 用全国线代替某省线。
- 用学校最低线代替专业录取线。
- 用旧高考数据直接套新高考专业组。
- 用没有来源的薪资/排名做结论。
